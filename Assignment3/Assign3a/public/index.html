📂 What is a Static Web Server?

A static web server delivers HTML, CSS, JavaScript, and image files directly to the user's browser without any server-side processing. It’s fast, lightweight, and great for small websites or testing purposes.

🛠️ Built With

  • Node.js
  • Core Modules: http, fs, url, path

🚀 Features

  • Serves static files like HTML, CSS, JS, images
  • Handles 404 errors gracefully
  • No external libraries required
  • Simple and minimal setup
